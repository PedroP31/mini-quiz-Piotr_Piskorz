O projekcie

To jest moja aplikacja „Mini Quiz”.
Działa tak, że można rozwiązać prosty quiz z pięcioma pytaniami, każde pytanie ma trzy odpowiedzi (A, B, C).

Co można robić:

-Kliknąć ROZPOCZNIJ QUIZ, żeby zacząć

-Wybrać odpowiedź na pytanie

-Wynik się aktualizuje po każdym pytaniu

-Na końcu pojawia się komunikat: „Koniec quizu! Twój wynik: X / 5”

-Można zresetować quiz przyciskiem RESETUJ QUIZ i zagrać jeszcze raz

Interfejs jest prosty:

-Wszystko ułożone pionowo, wyśrodkowane

-Tytuł największą czcionką, przyciski i napisy wyśrodkowane

-Marginesy między elementami min. 10dp

-Przyciski mają różne kolory, żeby łatwo było klikać

Jak uruchomić

1.Skopiuj projekt na komputer:

git clone https://github.com/PedroP31/mini-quiz-Piotr_Piskorz.git

2.Otwórz projekt w Android Studio:

-Kliknij „Open an existing Android Studio project”

-Wskaż folder projektu

3.Uruchom aplikację:

-Włącz emulator Android lub podłącz telefon

-Kliknij zieloną strzałkę Run

-Poczekaj aż aplikacja się załaduje

4.Jak korzystać:

-Widać tytuł i przycisk ROZPOCZNIJ QUIZ

-Kliknij go, żeby zacząć

-Wybierz odpowiedź na każde pytanie

-Wynik aktualizuje się automatycznie

-Na końcu wyskakuje komunikat z wynikiem

-Możesz kliknąć RESETUJ QUIZ, żeby zagrać jeszcze raz
